# Peer-Group Beta Rechner – Case IV

Webapplikation zu Case IV (Kapitalkosten), Teilaufgabe 2c).

Eingabe von bis zu 10 Peer-Unternehmen (Equity-Beta und Finanzierungsverhältnis FK/GK), Berechnung des Asset-Betas der Peer-Group (Mittelwert und Median, β_FK = 0, Tax-Shield unsicher) und des hypothetischen Equity-Betas des Zielunternehmens bei Eingabe der Zielkapitalstruktur.
